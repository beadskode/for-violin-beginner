# 개요

바이올린 입문자를 위해 오선지 악보의 각 조별 **바이올린 1포지션 운지점**을 안내하는 사이트.
한국어 사용자를 타겟으로 하며, 모든 텍스트는 **한국어/영어 병기**를 기본으로 하고 우상단 언어 토글로 강조 언어를 전환한다.

- 스택: React + Vite + TypeScript (정적 SPA)
- 외부 음악 렌더링 라이브러리 없이 오선지/음표는 SVG로 직접 그린다.

---

# 레이아웃

전체 2행 구성. 1행은 2컬럼, 2행은 1컬럼.

- **데스크탑(가로)**: 1행 = 좌측(조 선택) + 우측(지판), 2행 = 오선지.
- **모바일(세로)**: 위에서부터 조 선택 → 지판 → 오선지 순으로 세로 스택.
- 모든 UI의 최대 여백(padding/margin)은 **16px**.
- 높이 800px대 모니터에서도 스크롤 없이 1행 전체가 보여야 한다.

## 1행 좌측 컬럼 — 조 선택
- 헤더: "조 선택 / Key".
- 조 버튼을 **오선지 조표 기준**으로 섹션 구분한다: 샵(♯) 조 / 다장조(조표 없음) / 플랫(♭) 조. (조 개수로 나누지 않는다.)
- 1포지션으로 연주 가능한 조를 버튼으로 표시한다. 각 버튼:
  - 높은음자리표가 있는 오선지에 해당 조의 조표를 그린 **SVG 이미지**.
  - 한글 조이름을 크게(예: 사장조), 영어 조이름을 작게(예: G Major) 병기.
- 장조/단조를 모두 포함한다. 같은 조표를 공유하는 장조·관계단조는 같은 섹션에 둔다.

## 1행 우측 컬럼 — 지판 / 운지점
- 헤더: 현재 선택된 조 이름(한/영 병기, 기본 다장조 / C Major).
- 바이올린 지판과 4개 현(G·D·A·E)을 SVG로 그린다.
  - 지판은 실제처럼 세로로 길고 가는 형태에 근사하되, 음이름 가독성을 위해 폭을 약간 넓힌다.
  - 16:9 영역 안에서 스크롤 없이 지판 전체가 보여야 한다.
- 1포지션의 **모든 운지점**을 항상 표시한다.
  - 선택한 조의 스케일에 포함되는 운지점은 **선명하게(활성)**, 포함되지 않으면 **흐리게(비활성)** 처리한다. 숨기지 않는다.
  - 각 운지점(스티커) 안에는 **한글 음이름**을 크게 표시한다. 손가락 번호(0~4)는 스티커 **테두리/배경 색**으로 구분하고, 색각이상 대비를 위해 숫자도 함께 표기한다.
  - 영어 음이름은 스티커 한글 아래 작게 표기한다.
  - 샵/플랫은 선택한 조의 조표 방향(샵 조→샵, 플랫 조→플랫)에 맞춰 음이름과 함께 표기한다.

## 2행 — 오선지
- 높은음자리표 오선지 한 마디에 4분음표 1개를 표시한다.

# 기능
- 우측 지판의 운지점을 클릭하면 해당 음 하나를 2행 오선지에 4분음표로 표시한다(누적이 아니라 교체).
- 좌측에서 조 버튼을 클릭하면 해당 조의 스케일 음만 우측 지판에서 활성화되고 나머지는 흐려진다. 우측 헤더의 조 이름도 갱신된다.
- 언어 토글로 한/영 강조 표시를 전환한다(병기는 유지).

# 데이터 모델 (1포지션)

개방현: G3 / D4 / A4 / E5. 1포지션은 개방현부터 4번 손가락(완전5도, +7반음)까지.

각 현의 반음 오프셋(0~7)별 음과 손가락 번호. 손가락 매핑: `[0,1,1,2,2,3,3,4]` (각 손가락의 낮은/높은 위치 포함).

| 오프셋 | G현 | D현 | A현 | E현 |
|---|---|---|---|---|
| 0 (0번) | G 솔 | D 레 | A 라 | E 미 |
| 1 (1번) | G♯/A♭ | D♯/E♭ | A♯/B♭ | F 파 |
| 2 (1번) | A 라 | E 미 | B 시 | F♯/G♭ |
| 3 (2번) | A♯/B♭ | F 파 | C 도 | G 솔 |
| 4 (2번) | B 시 | F♯/G♭ | C♯/D♭ | G♯/A♭ |
| 5 (3번) | C 도 | G 솔 | D 레 | A 라 |
| 6 (3번) | C♯/D♭ | G♯/A♭ | D♯/E♭ | A♯/B♭ |
| 7 (4번) | D 레 | A 라 | E 미 | B 시 |

- 운지점의 절대음(MIDI) = 개방현 MIDI + 오프셋. 오선지 위치 계산과 조 스케일 포함 여부 판정에 사용.
- 활성/비활성: 운지점의 음높이 클래스(pitch class)가 선택한 조 스케일에 포함되면 활성.
- 음이름 표기는 선택한 조의 조표 방향(샵/플랫)을 따른다.

# 조 데이터
- 5도권 기반으로 장조 + 관계단조 생성. 각 조: 으뜸음 pitch class, 모드(장/단), 조표(샵/플랫 + 개수), 한/영 이름.
- 섹션: 샵 조 / 다장조(C, 조표 없음) / 플랫 조.

# 손가락 색상 (잠정)
- 0번(개방): 회색, 1번: 파랑, 2번: 초록, 3번: 주황, 4번: 빨강. (색 + 숫자 병기)

# 접근성 / 기타
- 손가락은 색만이 아니라 숫자도 표기.
- 선택한 조는 URL 쿼리 또는 localStorage로 유지(선택 구현).
